# Инструкция по запуску

## Что понадобится (один раз)

### 1. Установить Python

Скачайте и установите Python 3.10 или новее:  
https://www.python.org/downloads/

**Важно:** при установке поставьте галочку «Add Python to PATH».

---

### 2. Получить Telegram API ключи

1. Откройте https://my.telegram.org и войдите через свой номер телефона
2. Нажмите **API development tools**
3. Заполните форму (название и описание — любые, например «Monitor»)
4. Скопируйте `api_id` (число) и `api_hash` (длинная строка)

---

### 3. Создать бота

1. Напишите в Telegram боту [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте название и username для бота
4. Скопируйте **токен** (выглядит так: `123456789:ABCdef...`)

---

### 4. Узнать свой Telegram ID

1. Напишите в Telegram боту [@userinfobot](https://t.me/userinfobot)
2. Он ответит вашим числовым ID (например: `123456789`)

---

### 5. Заполнить config.yaml

Откройте файл `config.yaml` и замените значения:

```yaml
telegram:
  api_id: 12345678          # ← вставьте ваш api_id
  api_hash: "abcdef..."     # ← вставьте ваш api_hash
  bot_token: "123:ABC..."   # ← вставьте токен бота
  your_user_id: 123456789   # ← вставьте ваш Telegram ID
```

Также добавьте каналы которые хотите отслеживать в раздел `channels`.

---

### 6. Установить зависимости

Откройте папку с проектом в терминале (командной строке) и выполните:

```
pip install -r requirements.txt
```

---

## Запуск

```
python main.py
```

При **первом запуске** программа попросит:
1. Введите ваш номер телефона (в формате +79991234567)
2. Введите код из SMS / Telegram
3. Если включена двухфакторная аутентификация — введите пароль

Это нужно только один раз — сессия сохранится в папку `sessions/`.

---

## Управление через бота

После запуска найдите своего бота в Telegram и отправьте ему:

| Команда | Что делает |
|---|---|
| `/help` | Показать справку |
| `/status` | Статистика: сколько проверено и найдено |
| `/channels` | Список отслеживаемых каналов |
| `/add @channel` | Добавить канал |
| `/remove @channel` | Убрать канал |
| `/keywords` | Список ключевых слов |
| `/add_kw слово` | Добавить ключевое слово |
| `/remove_kw слово` | Убрать ключевое слово |
| `/pause` | Приостановить мониторинг |
| `/resume` | Возобновить мониторинг |
| `/recent` | Последние 5 найденных заказов |

---

## Как добавлять каналы

Можно двумя способами:

**1. Прямо в боте** (без перезапуска):
```
/add @имя_канала
```

**2. В config.yaml** (до запуска):
```yaml
channels:
  - "@freelance_ru"
  - "@мой_канал"
```

---

## Как это работает

```
Telegram-каналы → [ваш аккаунт слушает] → фильтр по ключевым словам → бот присылает уведомление
```

Программа работает от имени вашего аккаунта (не бота), поэтому видит все публичные каналы на которые вы подписаны. Мониторить можно только те каналы, в которых вы состоите.

---

## Автозапуск (необязательно)

Чтобы мониторинг запускался автоматически при включении компьютера, создайте задачу в «Планировщике задач» Windows с командой:

```
python C:\путь\к\папке\main.py
```

---

## Частые вопросы

**Бот не присылает уведомления?**  
→ Убедитесь что вы написали боту /start хотя бы раз.

**Программа упала с ошибкой?**  
→ Проверьте правильность данных в config.yaml.

**Хочу следить за закрытым каналом?**  
→ Вступите в него со своего аккаунта, затем добавьте через /add.
